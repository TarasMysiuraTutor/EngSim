# 🧮 Інженерні Калькулятори

Професійні інструменти для інженерних розрахунків з багатомовним інтерфейсом.

## 📊 Категорії розрахунків

### 💪 Опір матеріалів
- Напруження, Балка на згин, Вал на кручення
- Болтове з'єднання, Деформація, Стійкість колони
- Пружина, Втомна міцність

### 💧 Гідравліка
- Розрахунок труб, Підбір насосу, Вентиляція
- Витікання з отвору, Гідравлічний опір
- Спорожнення резервуара, Кавітація, Місцеві втрати

### 🔥 Термодинаміка
- Теплопередача, Теплоізоляція, ККД циклу
- Ідеальний газ, Ентальпія, Ентропія
- Цикл Карно, Теплоємність

### ⚡ Енергетика
- Енергозбереження, Потужність обладнання
- Сонячна енергія, Вітрова енергія, Теплонасос
- Акумулятор, Трансформатор, ЛЕП
